【*】程序简介

-工程名称：出厂测试程序
-实验平台: 野火 GDH759IMK6 GD32开发板  
-MDK版本：5.36
-C COMPILER 版本：6.16
-ST固件库版本：V1.0.0

【 ！】功能简介：

完成底板测试任务
1、LED(1-2-3)
2、KEY(1-2)
3、RTC(RCU_LXTAL)
4、SDRAM
5、FLASH
6、电位器
7、BEEP
8、DEBUG串口
9、网口
10、屏幕
11、SD卡

【*】注意事项：
把蜂鸣器上的贴纸撕掉可以得到较好的声响效果。

【 ！】实验操作：
编译并下载程序到开发板

第一步：观察到上位机有（提示与输出表示debug串口正常）如：欢迎使用野火 GDH759IMK6-GD32开发板
第二步：等待网口测试,网口不通过则除电源灯全部都不亮，SD卡不通过则LED1和LED2长亮,测试通过后LED3不停闪烁
第三步：数秒后(通常2s以内),只要LED3不闪烁就是测试不过关--测试过程中LED3会不断闪烁,否则程序错误(按下NRST按键可复位并重新开始测试)
第四步：自动进行SDRAM、FLASH、RTC操作(过程非常短,灯闪即可进行下一步)
第五步：前面通过后，蜂鸣器响一下，开始初始化屏幕，屏幕到蓝色和LED3继续闪烁后
第六步：按下按键(KEY1对应LED1,KEY2对应LED2),正常情况按下按键后对应LED会亮,否则测试不通过。测试通过后即可进行下一步
第七步：逆时针旋转电位器到底再顺时针旋转到底（也可先顺再逆）。若初始状态已经在底，则只需要反方向旋转一次即可（若不确定就旋转两次）
第八步：数秒后(通常2s以内),LED1-LED2-LED3闪几下，蜂鸣器最后会响一下,表示全部测试通过，否则蜂鸣器长响（测试不通过）
