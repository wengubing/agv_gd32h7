【*】程序简介
-工程名称：基于RTX5的实时操作系统点灯
-运行平台: 野火 GDH759IMK6-GD32开发板  

-功能简介: 	LED1翻转，时间间隔1000ms;
			LED2基于消息队列中接收到的消息进行翻转，翻转间隔50ms，消息发送间隔2s
			LED3基于二值信号量翻转，二值信号量通过检测KEY1按键释放
/*********************************************************************************************/

程序框架介绍：
BSP--------------->硬件驱动层
APP--------------->上层应用层
User-------------->主函数
RTX5_...---------->RTX5系统相关源码
GD32H7XXFWLIB----->GD32官方固件库
CMSIS------------->GD32系统函数
Startup----------->启动函数

/*********************************************************************************************/
信号量、互斥量使用终结：

使用场景				 推荐机制				说明

中断通知任务			二值信号量		ISR释放，任务等待，实现快速响应

任务间一次性同步		二值信号量		一个任务等待另一个任务完成初始化

管理多个相同资源		计数信号量		如DMA缓冲区、硬件通道池

保护共享资源			互斥量			如串口、I2C总线，避免数据竞争

生产者-消费者模式		计数信号量		计数表示可用数据量


 // 声名信号量ID
osSemaphoreId_t initSem;
// 创建二值信号量，初始值为0
initSem = osSemaphoreNew(1U, 0U, NULL);

  // 释放信号量
 osSemaphoreRelease(initSem);

 // 等待initSem信号量释放（信号量ID，等待时间）
 osSemaphoreAcquire(initSem, osWaitForever);
